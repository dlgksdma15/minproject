Security Demo: XSS and SQL Injection

Cross-Site Scripting (XSS)

XSS 공격은 웹 애플리케이션에서 발생하는 가장 일반적인 취약점 중 하나입니다. 악의적인 사용자가 웹 페이지에 스크립트를 삽입하여 다른 사용자의 브라우저에서 실행되도록 할 수 있습니다.

SQL Injection

SQL Injection은 데이터베이스 쿼리에 사용자가 입력한 데이터를 직접 포함하여 발생하는 취약점입니다. 이를 통해 공격자는 데이터베이스를 조작할 수 있습니다.

UNION SQL Injection

Union SQL Injection은 데이터베이스 쿼리에 Union을 이용하여 추가적인 정보를 추출하는 취약점입니다. 공격자는 이를 통해 데이터베이스에서 민감한 정보를 탈취할 수 있습니다.
